# How to use the "MoveBeatmaps"
1.Execute main.exe.
2.Open the settings via the "Settings" button and select osu!/Songs folder via the "Select folder" button.
3.Press the "Save" button next to the "Select folder" button. Then, the setting will be saved.
4.Press the "Move beatmaps" button and all beatmaps in Download folder will be moved.

# Attention
1. Do not modify the contents of config.ini. It may stop working.

# About update
When the latest version is released, an update notification window will appear when the application is launched.
Follow the same procedure as for downloading to download the latest version and replace it with the current version.
